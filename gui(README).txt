Student Management System

This application is a simple Student Management System that allows users to:
- Log in with a username and password.
- Create a new account.
- Add, edit, delete, and toggle the attendance status of student names.
- View saved lists of student names.
- Save the list of student names to a file.

Files:
- GUI.py: The main Python script containing the implementation of the Student Management System.
- student_names.txt: A text file used to store the list of student names.

Usage:
1. Run the Practice Test.py script using Python.
2. Log in with a username and password or create a new account.
3. Once logged in, you can add, edit, delete, and toggle the attendance status of student names.
4. You can also view saved lists of student names.
5. Changes made to the list of student names are automatically saved to the student_names.txt file.

Dependencies:
- Python 3.12.1
- tkinter library (for GUI)
- datetime library (for date and time operations)
